## Steps to Contribute

1. **Fork this repository.**
2. **Pick an issue** labeled `good-first-issue` or `feature-request`.
3. **Create a branch**, write your code and tests.
4. **Submit a pull request (PR)** with a clear title and description of your changes.

2. **Clone your forked repository** to your machine:
   ```bash
   git clone https://github.com/Gogi2016/ToDoApp.git
   cd ToDoApp

Install dependencies:
npm install

Run tests:
npm test
